# comman-network-security-threats
**Common network security threats** are potential dangers like malware, phishing, and hacking attacks that can compromise the confidentiality, integrity, or availability of a network.
# Common Network Security Threats

##  Overview
This repository provides an overview of common network security threats, their impact on systems, and basic prevention techniques. It is useful for students, beginners in cybersecurity, and academic projects.

---

##  Objective
- Identify common network security threats  
- Understand how attacks occur in real-world networks  
- Learn basic prevention and mitigation techniques  
- Improve cybersecurity awareness  

---

##  Common Network Security Threats

### 1.  Malware
Malicious software designed to damage or disrupt systems.  
Types include viruses, worms, trojans, spyware, and ransomware.

---

### 2.  Phishing
Fraudulent attempts to steal sensitive information such as passwords and credit card details using fake emails or websites.

---

### 3.  DoS / DDoS Attacks
Attacks that flood a server or network with traffic, making it unavailable to users.

---

### 4.  Man-in-the-Middle (MITM)
Attackers secretly intercept communication between two parties to steal or modify data.

---

### 5.  SQL Injection
Malicious SQL code is inserted into input fields to access or manipulate databases.

---

### 6.  Password Attacks
Methods like brute force, dictionary attacks, or credential stuffing used to crack passwords.

---

### 7.  Eavesdropping (Sniffing)
Unauthorized monitoring of network traffic to capture sensitive data.

---

### 8. Zero-Day Exploits
Attacks that exploit unknown software vulnerabilities before a fix is released.

---

### 9.  Insider Threats
Security risks caused by trusted users who misuse their access intentionally or accidentally.

---

##  Impact of These Threats
- Data breaches  
- Financial losses  
- System downtime  
- Identity theft  
- Reputation damage  

---

##  Prevention Techniques
- Use firewalls and IDS/IPS systems  
- Regular software updates and patching  
- Strong passwords and multi-factor authentication (MFA)  
- Data encryption  
- Network monitoring tools  
- User awareness training  

---

##  Real-World Example
### WannaCry Ransomware Attack (2017)
- Affected 200,000+ systems worldwide  
- Exploited Windows vulnerability  
- Caused major disruption in healthcare and businesses  
- Demanded ransom in Bitcoin
